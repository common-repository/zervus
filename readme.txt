== Description ==

Dieses Plugin aendert die Begruessung in der Adminleiste oben rechts, nach dem Einloggen. Basiert auf <a href="https://wordpress.org/plugins/howdy-against-humanity/" target="_blank">Howdy Against Humanity</a>

== Installation ==

1. Upload `zervus` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
